Cygwin-installer

What it does: Installs latest Cygwin, includes your list of packages, and preconfigures profile for you. 1. Gets install.exe from Cygwin.com 1. Runs unattended install including custom list of packages 1. Inserts custom configuration (shared team config, your file replication location, etc.) What you do: Run Install-Cygwin.bat, then open Cygwin prompt to start working!
